HOW TO RUN STATISTIPARK.

Pre-requisites:
	1. Java 8 (or later) instaled.
	2. Oracle SQL instaled.
	3. 50GB free on the Hard Disk.
	4. 2GB RAM.

Steps:
	1. Connect to the database and run "Database Init.sql".
	2. Run StatistiPark. You can launch the .jar file or open the project in Eclipse and run it from there.
